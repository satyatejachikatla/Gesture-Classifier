# How to run:
Select the parameters by changing the file params.cfg.
The format is
\<Models directory\>
\<Training data directory\>
\<Test data directory\>
\<epochs\>
\<batch size\>
\<learning rate\>
\<boolean value: whether model should be reloaded\>
